# Workflows

Each file here is a Markdown SOP that defines how to accomplish a specific task.

## Workflow Template

```
# Workflow Name

## Objective
What this workflow accomplishes and when to use it.

## Inputs Required
- input_1: description
- input_2: description

## Tools Used
- tools/script_name.py — what it does

## Steps
1. Step one
2. Step two
3. Step three

## Expected Output
What success looks like and where the output goes.

## Edge Cases & Known Constraints
- Any rate limits, quirks, or failure modes discovered during use
```
